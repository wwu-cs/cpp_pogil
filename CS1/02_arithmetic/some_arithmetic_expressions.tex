    \item If the parentheses were removed from the C++ Expression in the second row
      so that it was {\tt 3+5/2}, what would its value be?
      \begin{solution}[0.5in]
        The new value would be $3+\frac{5}{2} = 3 + 2 = 5$
      \end{solution}
