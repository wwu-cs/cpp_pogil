# cs2_cpp_pogil

This folder contains weekly homework assignments for CPTR 242 at Walla Walla University. CPTR 242 focuses on data structures and algorithms.

Each subfolder contains various files used in the homework. Including a README.md file that provides a description of the requirements necessary to pass the automated tests in submitty.

## CPTR 242 Homework Assignments

* [Homework 2- Sorting](02_sorting)

* [Homework 6- Graphs](06-graphs)
